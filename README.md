## Подключение

### Способ 1 — Karing ⭐

1. Установите **Karing**
2. Скопируйте ссылку ниже
3. Откройте Karing → **Профили → Добавить → URL**
4. Вставьте ссылку и сохраните профиль
5. Выберите сервер и подключитесь
```
https://raw.githubusercontent.com/Signal11-hub/AlaraVpn/main/sub.txt
```
